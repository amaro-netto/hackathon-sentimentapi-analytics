Como Funciona

Este sistema classifica avaliações de usuários em três categorias principais:

  • Positiva: Avaliações com tom favorável, elogios e recomendações.
  • Negativa: Avaliações com críticas, insatisfação e reclamações.
  • Neutra: Avaliações factuais, sem forte tom positivo ou negativo.

O sistema analisa o texto utilizando processamento de linguagem natural para determinar o sentimento e a confiança da classificação.
